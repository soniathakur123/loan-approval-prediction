# Loan Approval Prediction Model

## Project Overview

This project uses Machine Learning to predict whether a loan application will be approved or rejected based on applicant details.

The dataset includes features such as applicant income, loan amount, credit history, education, and property area.

## Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Scikit-learn
* Jupyter Notebook

## Machine Learning Models Used

* Logistic Regression
* Decision Tree
* Random Forest

## Project Workflow

1. Data Loading
2. Data Cleaning and Handling Missing Values
3. Label Encoding for categorical features
4. Data Visualization
5. Train-Test Split
6. Feature Scaling
7. Model Training
8. Model Evaluation (Precision, Recall, F1 Score, Accuracy)

## Dataset

The dataset used in this project is stored in the `data` folder.
